# Playbook 01 — 宏观市场与事件影响

> 触发：大盘复盘、指数走势、融资余额、商品（油价 / 黄金 / 白银）、地缘 / 政策事件。

## 1. 必跑数据清单

| # | 数据 | 入口 |
|---|------|-----|
| 1 | 上证 / 深成 / 沪深300 / 创业板 / 科创50 / 恒生 / 恒生科技 / 纳指 / 标普 当日涨跌幅 + 近 N 日 | `fast-snapshot` + `fast-window` |
| 2 | 商品（黄金 / 白银 / 原油 / 煤炭）相关期货代码 | grep `assets_db/future.yaml` → `fast-snapshot` / `fast-window` |
| 3 | 融资余额 / 北向资金 | quant-buddy `confirmDataMulti` 验证字段是否存在；不存在 → §6 |
| 4 | 板块 / 行业涨跌排序 | `quant-standard` + `recipes/industry-aggregation.md` |
| 5 | 历次相似事件后 N 日表现 | `event-study.md`（必须先识别事件日） |

## 2. 非数据信息

- 当日 / 当周重大消息（要标来源 + 时点）；
- 政策原文链接；
- 海外 PMI / CPI / 联储讲话日历。

## 3. 输出骨架

```
【结论行】今天 / 本周 大盘 <方向>，主线 <赛道>，关键变量 <事件>。
【指数表】上证/深成/沪深300/创业板/科创50/恒科/纳指 当日 + 近 5/20 日
【板块异动】涨幅前 5、跌幅前 5（来自行业聚合）
【资金面】融资余额 / 北向 / 主力（仅 quant-buddy 实际有的字段）
【事件解读】对应板块映射，分短期 / 中期影响
【操作框架】仓位倾向（不直接给点位）
【风险】+【声明】
```

## 4. 特殊场景

- **"今天什么原因深 V"**：先用 `fast-window` 取分钟级（若 quant-buddy 提供）或当日开高低收 + 板块异动；解读必须标"基于盘后数据回看"，不要冒充实时盘中。
- **"明日油价预测"**：明确标注**不是 quant-buddy 数据，是基于历史区间统计 / 事件研究的概率推断**；无量化模型支撑 → §6。
- **"美伊局势对纳指影响"**：定性问题；可用 `event-study` 跑历史相似事件后纳指 N 日表现，并把结论限定在历史统计含义内。

## 5. 失败兜底特化

- "明日盘前关注消息"——一手新闻源能否取到？取不到不要编。
- "X 时间点之前的市场情况"——quant-buddy 历史数据应能覆盖；若指数代码没找到 → grep `assets_db/index.yaml` 再试一次；仍不行 → §6。
